Description
===========

node-selinux is an libselinux binding for [node.js](http://nodejs.org/).


Requirements
============

* [node.js](http://nodejs.org/) -- tested with v0.1.94
* libselinux

To build node-selinux:

	node-waf configure build

API Documentation
